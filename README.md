# Patientenverwaltung (C#)

Einfache Konsolenanwendung zur Verwaltung von Patienten.
Die Anwendung ermöglicht das Anlegen, Anzeigen, Bearbeiten, Suchen und Löschen von Patienten sowie das Speichern und Laden der Daten über eine JSON-Datei.

## Funktionen
* Patienten hinzufügen
* Patienten anzeigen
* Patienten bearbeiten
* Patienten suchen
* Patienten löschen
* Speicherung und Laden der Daten über JSON

## Technologien
* C#
* .NET
* JSON

*Dieses Projekt wurde als Lernprojekt zur Vertiefung von C# und .NET erstellt.*
